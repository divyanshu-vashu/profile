# LangGraph with MCP

Model Context Protocol (MCP) allows AI models to connect seamlessly to external datasources and APIs. By using LangGraph, we can build stateful, multi-agent systems where each agent exposes its specialized tools via MCP.

## Key Benefits
- **Decoupled Architecture**: Keep tools isolated from agent logic.
- **Dynamic Context**: Agents fetch what they need dynamically.
